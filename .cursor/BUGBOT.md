# Bugbot Review Rules

Flag these issues in Run Together changes:

- Missing migrations for schema changes.
- Service role usage in client code.
- Broad unrelated refactors.
- Overengineered MVP changes.
- Auth or session regressions.
- Run creation, list, or join-flow regressions.
- Participant access or visibility regressions.
- Geo, map, or autocomplete layout regressions.
- Notification or Telegram side-effect risks.
- Mobile PWA layout risks.
